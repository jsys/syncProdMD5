# syncProdMD5

Synchronise local et production avec controle MD5. 

Plus rapide et fiable qu'un transfert FTP par exemple.

## Installation

  1. Modifier la config en entête du fichier syncProdMD5.php
  2. uploader le fichier en prod à l'url indiqué dans la config
  3. Appelez le fichier en local pour afficher la différence entre le local et la prod
  4. Cliquez sur les liens pour uploader ou downloader les différences
